# PROJETO CADASTRO DE SÉRIES :clapper:

  Nesse projeto foi realizado a construção do Aplicativo de Séries em memória, passo a passo junto com  professor Eliézer Zarpelão. 
  Uma dica importante para quem está começando... Fazer passo a passo e ir implementando junto com o professor. O aprendizado é enriquecedor e muito mais produtivo.
  Consegui entender todo o projeto e incrementei o método de Atualização de Série, fornecendo a opção de alterar um item da série ou todos os itens. 
  Realizei também uma confirmação de Exclusão da série, sendo possível também restaurar a exclusão pela Atualização da série.
  Sinto que posso melhorar bem mais o código e simplificar, assim como tratar os erros, porém fiz algumas tentativas que não deram certo, mas desistir não é uma opção.
  Aos poucos vou progredindo nos projetos e irei entendendo cada vez mais para aprimorar meus projetos.
